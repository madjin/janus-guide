KB+MOUSE EDITING

Make sure editing mode is enabled in the escape menu and editing mode has been enabled for that room (this is up to the discretion of the room owner).

Action Command
Select Right click on object
Choose property Tab
Modify property W,A,S,D,Q,E
Accept change Left click
Cancel change Right click
Delete selected Delete or backspace
Copy selected Control + C
Paste selected Control + V
New object Right click blank area
New image Control + right click blank area
New text Shift + right click blank area
Snap amount 1,2,3,4
Save room Control + S
Reload room Control + R
Bookmark room Control + B

CHAT WINDOW COMMANDS

These commands can be entered into the chat window, your current settings can be found in the userid.txt file in your Janus VR folder, if you are unsure of changing any settings then you probably should leave them alone.

WARNING: If make changes that render Janus unusable you can remove settings.txt from your Janus VR directory to revert to default settings.

Action Command
#userid Change your name*
#sync Sync objects with other players
#eye_pos Change your view position
#eye_ipd Changes distance between eyes
#head_pos Changes your head position
#clearchat Clears the chatlog
#scale Scales your avatar
#pos Changes your position
#lighting_false Lighting true or false
#cull_face_back Cull face back, front or none
#head_src Url to head .obj / .dae
#head_mtl Url to head .mtl
#body_src Url to body .obj / .dae
#budy_mtl Url to body .mtl

*Your name should be the same as you use in the mumble chat.
